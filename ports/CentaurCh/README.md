# CentaurCh

`CentaurCh` is a standalone SuperCollider port of the traditional circuit path from the ChowDSP `ChowCentaur` plugin.

This first pass includes:

- input buffer
- traditional nonlinear gain stage
- tone stage
- output buffer
- output DC blocker

This first pass does not include the optional neural gain-stage model, plugin bypass crossfade, or explicit mono fold mode.
